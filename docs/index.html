Operaciones con strings

"Hola " + "mundo" // devuelve Hola mundo

Operaciones con números enteros

1 + 1 // devuelve 2
1 - 1 // devuelve 0
2 * 2 // devuelve 4
4 / 2 // devuelve 2
!-5 // devuelve 5
!5 // devuelve -5

Operaciones con números flotantes

1.5 + 1.5 // devuelve 3
1.5 - 0.5 // devuelve 1.0
.3 * 2.5 // devuelve 0.75
4.1 / 2.33 // devuelve 1.759656652
!-5.5 // devuelve 5.5
!5.5 // devuelve -5.5

Operaciones con booleanos

true == true // devuelve true
false != true // devuelve true
!true // devuelve false
!!false // devuelve true

Operaciones condicionales

5 == 5 // devuelve true
4 < 19 // devuelve true
50 > 0.4 // devuelve true
2 != 2 // devuelve false

Definir variables

set num = 34
set num // Puedes definir una variable sin valor
num = 34 // Y luego puedes definirle el valor de esta forma
set num: Integer = 34 // También puedes definir la variable con un tipo ya definido

Declaración while

...

endwhile

// Declaración Do While
do // este ciclo while no lleva "endwhile" al final
...
num++ // podemos sumar un número a una variable así
